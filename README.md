# WordSplitter — сортировка слов по файлам по первой букве

Программа на C++, которая читает файл `words.txt`, удаляет ведущие пробелы у каждой строки и автоматически распределяет слова по отдельным файлам в зависимости от первой буквы.  
Например: все слова на **А** попадут в `A.txt`, на **Б** — в `Б.txt` и т.д.

## 🚀 Как работает программа

1. Открывается файл `words.txt`.
2. Каждая строка очищается от пробелов в начале.
3. Определяется первая буква слова.
4. Если буква изменилась — предыдущая группа слов записывается в файл `<Буква>.txt`.
5. После обработки всех строк создаётся последний файл.
6. На выходе получается набор файлов: `A.txt`, `Б.txt`, `В.txt` и т.д.

Фрагмент из кода, отвечающий за запись файлов:

> `fileWrite << oneSimbolWord;`  
> *(из файла main.cpp)*

```
project/

```
## 🛠️ Компиляция и запуск

### Через g++
```bash
g++ main.cpp -o splitter
./splitter
Через CMake (если используешь)
bash
mkdir build
cd build
cmake ..
cmake --build .
./splitter
```

📄 Формат входного файла
words.txt должен содержать слова, по одному на строку.
Пробелы в начале строки автоматически удаляются.

Пример:

Код
   яблоко
банан
   апельсин
📦 Результат работы
После запуска появятся файлы:

Код
A.txt
Б.txt
В.txt
...
Каждый файл содержит слова, начинающиеся на соответствующую букву.
